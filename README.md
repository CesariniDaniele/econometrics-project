# Econometrics Project – Job Training Program Analysis

## Viewing the Analysis

The analysis notebooks were rendered into HTML reports using Quarto.

To view the complete output (tables, figures, regression results, and discussion), it is sufficient to open:

- `notebooks/01_eda.html`
- `notebooks/02_models.html`

directly in a web browser.

The corresponding `.qmd` files contain the original Quarto notebooks with the R code used to generate the analysis.

## Directory Description

data/:

Contains the original dataset used throughout the analysis.
jtrain3.xlsx --> dataset used for the econometric analysis.

notebooks/:

Contains the Quarto notebooks (.qmd) used for the project, together with the rendered HTML reports and their associated folders.

01_eda.qmd --> 
Contains the exploratory data analysis (EDA)

02_models.qmd -->
Contains the econometric analysis, including:
- baseline OLS model;
- extended regression with controls;
- interaction effects;
- model diagnostics;
- robust standard errors;
- heterogeneous treatment effects;
- causal interpretation and policy discussion.

Each .qmd notebook is rendered into an HTML report:

01_eda.html
02_models.html

_files/:

Automatically generated by Quarto during rendering. They contain all the resources required by the HTML reports.
